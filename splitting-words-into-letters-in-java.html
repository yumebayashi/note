splitting words into letters in Java

金 13 5月 2016

  • -java7
"hoge".split("");
// ["", "h", "o", "g", "e"]
"hoge".split("(?<=.)");
// ["h", "o", "g", "e"]
  • java8-
"hoge".split("");
// ["h", "o", "g", "e"]
